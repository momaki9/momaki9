Concept 2: Built-in Methods


getElementById

Associates an HTML element by it's ID with a variable (firstChild)

var firstChild = document.getElementById("first-child");

Subsequently, the variable associated with the HTML element can be styled using javascript as follows:

firstChild.style.color = "red"

Similarly, this could be applied to child elements within the HTML that is associated with a variable. In the example below, we are changing the style (font size) of the first child of the selected element.

var articlesDiv = document.getElementById('articles'); articlesDiv.children[0].style.fontSize = '50px';


.querySelector, .querySelectorAll, and .setAttribute

You can access multiple elements using .querySelectorAll(), whereas .querySelector can be used to select an element by ID or by using it's class. If selecting for ID, be sure to use the # before the ID. If selecting a class, a . is used (similar to CSS selectors).

.setAttribute is used with the variable to change its style. The syntax used is similar to how style is coded in CSS. See example below.

var divTags = document.querySelectorAll("div");

var pTags = document.querySelectorAll("p");

var imgEl = document.querySelectorAll("img");

Acess element using it's ID: var changeP = document.querySelector("#change2");

This sets first pTags to have a font-size of 40px: pTags[0].setAttribute("style", "font-size: 40px;");

Sets changeP to have multiple style attributes: changeP.setAttribute("style", "font-size: 25px; font-weight: bold; text-decoration:underline; ");

Sets image source of the first image: imgEl[0].setAttribute("src", "./assets/images/image_1.png");

Adds size and width styling to image: imgEl[0].setAttribute("style", "width:50%");

Loops through divTags to set each one to have the color blue and the font size of 30px: for (var i = 0; i < divTags.length; i++) {

[i].setAttribute("style", "color:blue; font-size: 30px");}


.textContent

This can be used to input text within an HTML element. Note that .querySelector is used here with a class (.sites).

var sitesEl = document.querySelector(".sites");

site1El.children[0].textContent = "Site 1";

Reminder, the specific child element can be selected. In this case, .setAttribute is used to provide the URL:

site1El.children[1].setAttribute("href", "https://google.com");


.appendChild and .createElement

New elements can be created with default styling using .createElement.

var tagName = prompt("Please enter an HTML Tag (ex. h1, h2, p, div):", "enter tag");

var tag = document.createElement(tagName);

Subsequently, .textContent can be used to define the text inside the element created.

tag.textContent = "This was made via a prompt. It's a " + tagName + " .";

Finally, append child actually outputs the text + element in the 'parent' element as specified (body in this example)

document.body.appendChild(tag);


setInterval() and clearInterval()

The setInterval() method, repeatedly calls a function or exectures a block of code with a fixed time delay between each call. This method returns an interval ID which uniquely identifies the interval, so you can remove it later by calling clearInterval().

The example below displays a timer counting down from 10 seconds and after that an image would be displayed:

Selects element by class

var timeEl = document.querySelector(".time");

Selects element by id

var mainEl = document.getElementById("main");
var secondsLeft = 10;

Function to create and append colorsplosion image

var timeInterval = setInterval(function() {
secondsLeft--
timeEl.textContent = secondsLeft + " seconds left till colorsplosion!";
if (secondsLeft === 0) {
clearInterval(timeInterval);
sendMessage(); }
}, 1000);
function sendMessage() {
timeEl.textContent = " ";
var imgEl = document.createElement("img");
imgEl.setAttribute("src", "images/image_1.jpg");
mainEl.appendChild(imgEl);}


Concept 3: Event Listener


.addEventListener

The .addEventListener method of the EventTarget interface sets up a function that will be called whenever the specified event is delivered to the target.

Common targets are Element, or its children, Document, and Window, but the target may be any object that supports events.

In the example below, the function would be triggered by the event of a click. Syntax:

variable.addEventListener("click", function () {} )


Concept 4: Preventing Default Behavior


.preventDefault()

The preventDefault() method of the Event interface tells the user agent that if the event does not get explicitly handled, its default action should not be taken as it normally would be.

The event continues to propagate as usual, unless one of its event listeners calls stopPropagation() or stopImmediatePropagation(), either of which terminates propagation at once.

Syntax:

function functionName(event) { event.preventDefault(); }
